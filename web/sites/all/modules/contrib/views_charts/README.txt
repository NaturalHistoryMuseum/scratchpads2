-- SUMMARY --

Views Charts Provides a "charts" style output for Views module so you can render
result-set not just as text (list, tabular) but as pie-chart, bar-chart,
scatter-plot etc.

For a full description of the module, visit the project page:
  http://drupal.org/project/views_charts

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/views_charts


-- REQUIREMENTS --

* Views 2 - <http://drupal.org/project/views>

* Charts and Graphs - <http://drupal.org/project/charts_graphs>


-- SUGGESTED MODULES --

* Views GroupBy - <http://drupal.org/project/views_groupby>


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* There is no user configuration for this module. All settings are available
  through Views UI.

-- UPGRADING --

* As Views Charts is highly dependent in the Charts and Graphs module you should
  always be sure that you have the most up to date version of Charts and Graphs
  when upgrading the Views Charts.
